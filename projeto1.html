Sistema de Gerenciamento

Concluído: Janeiro 2023 Desenvolvimento Web

Sobre o Projeto

Um sistema completo de gerenciamento desenvolvido para otimizar processos internos de empresas. A plataforma inclui dashboard administrativo, módulo de relatórios e integração com APIs de terceiros.

Tecnologias Utilizadas

  • React.js para o frontend
  • Node.js com Express para o backend
  • MongoDB para armazenamento de dados
  • Chart.js para visualização de dados
  • JWT para autenticação

Desafios e Soluções

O maior desafio foi implementar a sincronização em tempo real dos dados entre diferentes módulos. Resolvi utilizando WebSockets para manter uma conexão persistente entre o cliente e o servidor.
